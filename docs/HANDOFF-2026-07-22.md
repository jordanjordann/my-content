## Product, in one paragraph

Content analysis tool for an **Indonesian social-media/marketing agency** (the user is the agency, not the creator). Paste an Instagram Reel or YouTube Short → ScrapeCreators fetches metadata → video downloaded → Gemini watches and analyses → stored. The destination is a **content brief generator**: the user names a topic, and the system generates a detailed brief for one video, written in a specific creator's style learned from ≥5 of their analyzed videos. Explicitly **not** a calendar or scheduler. Output language Indonesian; UI stays English. Desktop dashboard.

## Orchestration lesson — this cost real time

**Continuing an agent via follow-up message spawns a fresh lineage** that loses both specialist identity and `isolation: "worktree"`. This caused:
- Two rounds of duplicate tickets (#55/#56, then #75/#76) requiring reconciliation passes
- Several blocked writes where agents correctly refused to work around the guard
- One PR (#61) that was a strict subset of another (#62)

**Brief once, completely. Spawn fresh with `isolation: "worktree"` rather than amending in increments.**

Also: agents hit account session limits twice, killing 3–5 in flight each time. Recovery was clean **because work lived in git worktrees, not agent memory**. Keep that property.

Stale worktrees accumulate — 27 at peak. `git worktree list` + remove those whose PRs are merged (squash-merge means `merge-base --is-ancestor` won't detect them; use `gh pr list --state merged` instead).
