// Design Note: Empty state differentiation

This is the "no matching results" empty state — triggered by filters returning zero rows. Contrast with the "no analyses exist" state (shown when the database is empty), which should have a different message, icon, and a "New Analysis" CTA button.

Design Annotations

Spacing

  • gap-2 (8px) between all filter controls
  • gap-1.5 (6px) between media type chips
  • Control height: h-9 (36px) standard, h-8 (32px) for chips
  • Page padding: p-6 (24px)
  • Table cell padding: p-3 (12px)
  • 8px grid rhythm throughout

Colors

  • Active bg: oklch(0.55 0.24 260) (light), oklch(0.68 0.18 255) (dark)
  • Active text: white on both modes
  • Border: CSS var --border synced with theme
  • Muted hover: bg-muted/50
  • Score colors: green ≥7, yellow ≥5, red <5
  • Clear all: indigo text with underline, indigo hover

Component Decisions

  • Platform: Segmented toggle — single-select radio group (All | IG | YT)
  • Media type: Multi-select chips — any combination, OR logic
  • Account: Select dropdown — falls back to "All accounts"
  • Sort: Select dropdown — 4 options, "Newest" default
  • Clear all: Right-aligned text link, hidden when no filters active
  • All controls use shadcn CSS variables for theme consistency

Accessibility

  • role="toolbar" on filter container
  • aria-pressed on media type chips
  • role="radiogroup" + aria-checked on platform toggle
  • Keyboard navigation via standard button/tab order
  • Contrast ratio: ≥4.5:1 for all text on backgrounds
  • Touch targets: minimum 32-36px height

Behavior

  • Instant filter application — no "Apply" button
  • URL search params: ?account=x&platform=ig&mediaType=reel,short&sort=newest
  • Tabular-nums for all score columns (prevents width jitter)
  • Table rows: hover:bg-muted/50 transition
  • Framer Motion for row entrance animations in production

Empty States

  • "No matching results": filter icon, "Clear all filters" button
  • "No analyses exist": different message, "New Analysis" CTA
  • Footer shows "X of Y" count with hidden count when filtered
  • Active filter badges row for visual reinforcement (optional)
  • Persistent state: URL params survive page refresh
